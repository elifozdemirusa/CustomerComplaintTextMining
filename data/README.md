# Customer Complaint Text Mining for Financial Service Insights

This project applies Natural Language Processing (NLP) and Machine Learning techniques
to analyze customer complaint data from financial services.

The objective is to extract actionable insights using both supervised
and unsupervised learning models.

---

## Key Techniques
- Text preprocessing and feature engineering (TF-IDF)
- Supervised classification (Logistic Regression, Random Forest, kNN)
- Unsupervised learning (K-Means, LDA)
- Model comparison using a unified train/test split
- Business interpretation of model results

---

## Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- NLP libraries
